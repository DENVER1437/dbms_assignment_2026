## 🚀 Deployment

### Deploy to Vercel

1. Go to [vercel.com](https://vercel.com) → **New Project**
2. Import your GitHub repo
3. Set **Root Directory** to `frontend`
4. Framework Preset: **Other**
5. Click **Deploy**

---

## 🖥️ Run Locally

### Frontend

cd frontend
python -m http.server 8080
